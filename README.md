# Micepad CLI

Command-line interface for the [Micepad](https://micepad.co) event management platform.

## Install

```bash
curl -fsSL https://micepad.co/install-cli | bash
```

## Usage

```bash
micepad login             # Authenticate with Micepad
micepad events list       # List your events
micepad events use SLUG   # Set active event
micepad pax list          # List participants
micepad help              # See all commands
```

See the [full command reference](https://micepad.co/docs/cli) for more.
